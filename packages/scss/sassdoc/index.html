    mixins

    mixins

    device

    @mixin device($deviceName: desktop) { ... }

    Description

    This mixin is used to apply css based on device, device name should be passed as argument.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $deviceName

    accepted values are 'mobile', 'tablet', 'desktop', 'large-desktop'

    		Stringdesktop

    Example

    apply styling by defined devices.

    .body {
font-size: 16px;

 @include device("mobile) {
   font-size: 18px;
 }
}

    Throws

    • No #{$deviceName} device found in $devices map.

    mobile

    @mixin mobile() { ... }

    Description

    This mixin is used to apply css for a mobile device.

    Parameters

    None.

    Example

    apply styles for screen < 32rem 512px.

font-size: 16px;

 @include mobile {
   font-size: 18px;
 }
}
